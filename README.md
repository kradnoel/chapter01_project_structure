## Set up

### Requirements]

- [Flutter](https://flutter.dev)


### Local development

1. Clone this repository and `cd` into it

```bash
git clone https://github.com/kradnoel/chapter01_project_structure.git
cd chapter01_project_structure
```

2. Run the project

```bash
flutter run
```
